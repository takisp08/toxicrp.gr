Errors

1.Failed to getinfo server after 3 attempts: Εάν δεν είστε στην Ελλάδα αυτός είναι ο λόγος, για να λυθεί αυτό θα πρέπει να επικοινωνήσετε με κάποιον Head Admin,εαν είστε στην Ελλάδα δοκιμάστε να κάνετε relog το fivem.

2.Unable to find SteamID, please relaunch FiveM with steam open or restart FiveM & Steam if steam is already open, για να λυθεί αυτό θα πρέπει να σιγουρευτείτε πως έχετε ανοιχτό το Steam.

3.Connection failded Obtaining configuration from server failed(timed out), δοκιμάστε να κάνετε restart το fivem και μπείτε με f8 και την ip του server
